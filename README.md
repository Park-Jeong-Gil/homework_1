* ### 실행 방법

터미널(cmd)에서 해당 프로젝트의 **src** 경로 진입 후 **'live-server'** 입력.

혹은, **src** 폴더 내에 **'index.html'** 을 인터넷 브라우저로 실행.
  
  
<br/>
